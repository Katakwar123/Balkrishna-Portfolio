# 🎨 Balkrishna Katakwar - Portfolio Website

Modern, animated portfolio website built with React, Tailwind CSS, and Framer Motion.

## 🚀 Tech Stack
- React 18 + Vite
- Tailwind CSS + Framer Motion
- Node.js + Express (Backend API)

## 📦 Quick Start

### Frontend
```bash
cd client
npm install
npm run dev
```

### Backend
```bash
cd server
npm install
npm run dev
```
